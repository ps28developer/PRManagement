# PR Review Application - Frontend

The interactive user interface for the PR Review Application, built with Vite and React.

## 🛠️ Prerequisites
- The backend server must be running.

## ⚙️ Installation

1.  **Install Dependencies:**
    ```bash
    npm install
    ```

2.  **Run Development Server:**
    ```bash
    npm run dev
    ```
    The app runs on [http://localhost:5173](http://localhost:5173).

## 🔑 Default Login Credentials (after seeding)
- **Admin**: `admin@example.com` / `admin123`
- **Employee**: `employee1@example.com` / `password123`
- **Lead Developer**: `lead1@example.com` / `password123`

## 🎨 Features
- **Dashboard**: High-level overview based on user role.
- **Raise PR**: Intuitive form to submit pull requests.
- **Review PRs**: Structured feedback system with severity tracking.
- **History**: lead developers can see their previous approvals.
